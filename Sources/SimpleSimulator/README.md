# Canavs API Basics

**Clear Rectangle**

[Reference](https://developer.mozilla.org/en-US/docs/Web/API/CanvasRenderingContext2D/clearRect)

```json
{
    "operation": "clearRect",
    {
        "x": 0,
        "y": 0,
        "width": 0,
        "height": 0
    }
}
```
