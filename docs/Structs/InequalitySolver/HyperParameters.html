HyperParameters

public struct HyperParameters

This struct is just a container for the hyper parameters that can be used to customize the behavior of the solver.

The default values of fairly aggressive, so will find pretty precisely the minimum. For real time applications, many of the iteration maximums can be heavily restricted.
